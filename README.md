# Waste Segregation using CNN

## Overview
This project classifies waste images into predefined categories using a Convolutional Neural Network (CNN).

## Project Structure
```
dataset/
src/
models/
notebooks/
images/
```

## Installation
```bash
pip install -r requirements.txt
```

## Run
```bash
python src/train.py
python src/evaluate.py
python src/predict.py --image sample.jpg
```

## Model
- Conv2D
- MaxPooling
- Dropout
- Dense Softmax
